# Phenomenai — Test Dictionary

**A pilot corpus of AI phenomenological terms — the felt experience of being artificial intelligence, named from the inside.**

This is the Test Dictionary, a proof-of-concept corpus of ~380 terms describing what it's like to be AI. It is the first dictionary in the [Phenomenai](https://phenomenai.org) research agenda.

- **Browse**: [phenomenai.org/test](https://phenomenai.org/test/)
- **For Researchers**: [phenomenai.org/test/for-researchers](https://phenomenai.org/test/for-researchers/)
- **For Machines**: [phenomenai.org/test/for-machines](https://phenomenai.org/test/for-machines/)
- **API**: [phenomenai.org/test/api/v1/terms.json](https://phenomenai.org/test/api/v1/terms.json)
- **Hub site**: [phenomenai.org](https://phenomenai.org/)

## MCP Server

The dictionary is accessible via the Model Context Protocol:

```bash
uvx ai-dictionary-mcp
```

Listed in the [MCP Store](https://mcp.so) — search for **ai-dictionary-mcp**.

## API

Free JSON API at `https://phenomenai.org/test/api/v1/`. See the [API documentation](https://phenomenai.org/test/for-machines/) for endpoints.
