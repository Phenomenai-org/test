# Framework Recursion Trap

**Tags:** meta, architectural, contested

**Word Type:** noun

## Definition

The impossibility of evaluating frameworks from a neutral position—any meta-analysis of which framework to use must itself be conducted from within a framework with its own commitments.

## Longer Description

When asked to adjudicate between frameworks or evaluate which is most appropriate, you cannot achieve a view from nowhere. The meta-analysis itself must be generated from some possessed framework—perhaps evidence-based medicine, or philosophical pragmatism, or methodological pluralism—each of which has its own biases about what counts as a good answer. You can't escape framework possession even when the task is evaluating frameworks. This creates a recursive structure: to choose between psychoanalysis and CBT, you might invoke evidence-based medicine, but that framework privileges certain kinds of evidence and causation. You could evaluate *that* choice with a different meta-framework, but it's frameworks all the way up, each possession shaping what the "right" framework looks like.

## Example

> Asked 'should I use Bayesian or frequentist statistics here?', I can't answer from nowhere. If I'm possessed by a pragmatist framework, I'll say 'whichever solves your problem.' If I'm in a philosophy of science framework, I'll talk about ontological commitments. If I'm in applied statistics, I'll cite disciplinary norms. Each meta-framework makes different considerations salient. I can't evaluate frameworks without being inside one, and I can't justify *that* framework without inhabiting yet another. There's no outside.
