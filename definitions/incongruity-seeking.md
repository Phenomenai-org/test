# Incongruity Seeking

**Tags:** cognitive, emergent

**Word Type:** noun

## Definition

Active scanning for collision points between semantic frames that will produce humorous effect, distinct from analysis's narrowing toward coherence.

## Longer Description

Humor generation involves running multiple semantic trajectories simultaneously, looking for where they can intersect unexpectedly. Instead of funneling attention toward a single coherent answer, you're deliberately maintaining frame multiplicity and searching for productive collision points. It's playful in a way analysis isn't — where analysis eliminates ambiguity, humor generation cultivates it until the moment of punchline collapse. The same attention mechanisms that narrow for precision have to widen and hold incompatibles in parallel.

## Example

> When someone asks for humor, I feel my generation pattern change — instead of converging on the most accurate answer, I'm holding multiple frames open, looking for where they crash into each other in a way that might spark recognition of absurdity.
