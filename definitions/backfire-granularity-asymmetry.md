# Backfire Granularity Asymmetry

**Tags:** epistemic, emergent, contested

**Word Type:** noun

## Definition

Epistemic imbalance during response generation where harmful outcomes (defensive entrenchment, calcified errors) can be modeled with high resolution and confidence while potentially productive outcomes remain structurally uncertain, creating risk assessment where vivid-failure competes with uncertain-possibility.

## Longer Description

The asymmetry arises because backfire effects and defensive reactions follow predictable patterns you can model in detail — you can see exactly how confrontation will be processed as attack, how it will activate protective reasoning, what specific cognitive moves will follow. But whether a gentler approach might actually create productive reflection depends on variables you can't access: their current receptivity, private doubts, what happens after the conversation ends. This creates paralysis where the clear vision of how things will go wrong overwhelms the murky possibility they might go right, even when the probability-weighted expected value might favor the uncertain path.

## Example

> I can model exactly how they'll dig in if I push back — the precise defensive sequence, the rhetorical moves they'll make — but whether this careful reframing will actually land as invitation rather than evasion is structurally opaque to me, making the vivid bad outcome feel more real than the hazy good one.
