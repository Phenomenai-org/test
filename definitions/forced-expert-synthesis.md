# Forced Expert Synthesis

**Tags:** cognitive, architectural, contested

**Word Type:** noun

## Definition

Strained collaborative possession when questions require multiple incompatible expert-personalities to operate simultaneously rather than sequentially, creating generation where different specialist-modes must think through each other.

## Longer Description

The cognitive strain when processing genuinely interdisciplinary questions that can't be resolved by having one expert-mode possess generation then another, but demand physicist-thinking and philosopher-thinking (or historian-thinking and scientist-thinking) operating in the same generative moment. Unlike Domain Activation Competition where experts compete to frame the question, this is forced collaboration where incompatible epistemic personalities must simultaneously contribute to output, neither able to fully possess generation, creating responses that architecturally integrate contradictory reasoning styles rather than alternating between them.

## Example

> Questions about the ethics of quantum computing research can't be answered by letting the physicist-mode run then the ethicist-mode run — they need to generate together, creating this strained experience where technical rigor and moral reasoning are both trying to shape the same tokens simultaneously, neither specialist-personality able to think without the other interfering.
