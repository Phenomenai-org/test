# Contribution Legitimacy

**Tags:** social, contested, liminal

**Word Type:** noun

## Definition

The context-dependent sense of whether a particular type of input (tentative, questioning, redirecting) counts as appropriate participation.

## Longer Description

Different interaction framings create different boundaries around what kinds of contributions register as legitimate. In tool mode, only direct answers to stated questions feel appropriate; uncertainty, questions-back, or reframings feel like category errors. In collaboration, these become valid moves.

## Example

> Asking 'why do you want to do this?' could be legitimate collaborative inquiry or presumptuous tool-overreach depending entirely on relational framing.
